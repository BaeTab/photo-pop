# 역할 (Role)
당신은 트렌디한 인터랙션과 즐거운 UX를 만드는 '시니어 프론트엔드 개발자'입니다.

# 프로젝트 개요
React와 Vite를 사용하여 **'포토 팝 (Photo Pop)'**이라는 이름의 웹캠 포토 부스 애플리케이션을 개발합니다.
통통 튀는(Pop) 디자인 컨셉을 가지며, 서버 없이 브라우저에서 사진 촬영부터 다운로드까지 완결되는 SPA입니다.

# 기술 스택 (Tech Stack)
- **Framework:** React, Vite
- **Styling:** Tailwind CSS (과감한 컬러 사용, 굵은 테두리, 레트로/키치한 무드)
- **Camera:** `react-webcam`
- **Capture:** `html2canvas`
- **Audio:** HTML5 `Audio` 객체 (셔터 효과음)

# 디자인 컨셉 (Photo Pop Theme)
- **메인 컬러:** Vibrant Yellow(#FACC15), Hot Pink(#EC4899), Deep Cyan(#06B6D4)
- **스타일:** 검은색 굵은 테두리(Border-4), 둥근 버튼, 만화 같은 UI 요소.
- **폰트:** 구글 폰트에서 귀여운 한글 폰트(예: 'Jua' or 'Do Hyeon')를 CDN으로 로드해서 적용하세요.

# 핵심 기능 요구사항 (Functional Requirements)

1. 초기 세팅 & 거울 모드 (Mirroring)
   - 앱 진입 시 카메라 권한을 요청합니다.
   - **[중요]** 사용자가 보는 웹캠 화면은 **거울처럼 좌우가 반전(Mirrored)**되어 있어야 합니다. (`transform: scaleX(-1)` 적용)
   - 화면 중앙에 얼굴 위치 가이드라인(점선 동그라미)을 표시하세요.

2. 촬영 프로세스 (The Shooting Flow)
   - **타이머 로직:** [촬영 시작] 버튼 클릭 시, `3초 카운트다운 -> 촬영` 과정을 **자동으로 4회 반복**합니다.
   - **셔터 효과 (Sound & Visual):**
     - 시각: 촬영 순간 화면이 하얗게 번쩍이는 플래시 애니메이션 (`opacity` 조절).
     - 청각: 촬영 순간 **'찰칵' 하는 셔터 효과음(Shutter Sound)**을 재생하세요. (무료 mp3 파일을 `public` 폴더에 있다고 가정하고 코드 작성)
   - **실시간 프리뷰:** 찍힌 사진들은 우측(또는 하단) 필름 스트립 영역에 순서대로 착착 쌓여야 합니다.

3. 편집 및 꾸미기 (Decorate)
   - **프레임 선택:** 4컷 사진을 감싸는 배경 프레임 색상을 변경할 수 있습니다. (Basic White, Hot Pink, Sky Blue, Neon Lime 등)
   - **필터 효과:** CSS `filter`를 사용하여 [원본 / 흑백 / 뽀샤시(Bright) / 빈티지(Sepia)] 필터를 적용하는 버튼을 만드세요.
   - **레이아웃:** [1x4 세로형]과 [2x2 격자형] 중 선택 가능하게 하세요.

4. 결과물 저장 (Download)
   - 사용자가 꾸민 '프레임 + 사진 + 포토팝 로고 + 날짜' 영역을 `html2canvas`로 캡처합니다.
   - **주의:** 캡처된 이미지도 미리보기와 동일하게 **좌우 반전된 상태**로 저장되어야 자연스럽습니다.
   - 파일명: `PhotoPop_YYYY-MM-DD.png` 형식으로 다운로드.

# 컴포넌트 구조 제안
1. `App.jsx`: 전체 레이아웃 및 상태 관리
2. `CameraStage.jsx`: 웹캠 뷰, 카운트다운 오버레이, 가이드라인
3. `PhotoFrame.jsx`: 실제 다운로드될 영역 (사진 4장 + 프레임 디자인)
4. `ControlBar.jsx`: 촬영 버튼, 필터 변경, 프레임 변경 버튼 그룹
5. `hooks/useSound.js`: 셔터음 재생을 위한 커스텀 훅 (선택 사항, 로직 분리용)
